## 关于实验

会有一个课程项目，需要小组合作，既有编码也有文档的需求。注意选好队友，小组合理分工，记得不要拖延！

update 2021：因为其他课程（OS、DB）实验密集，所以删掉了软工的课程项目，考试比重加大。
